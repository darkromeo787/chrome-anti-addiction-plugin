# 使用說明

## 安裝未封裝擴充功能

1. 開啟 `chrome://extensions`。
2. 開啟開發人員模式。
3. 點選「載入未封裝項目」。
4. 選擇本專案資料夾。
5. 如果需要快速查看狀態，可以把擴充功能固定到工具列。

## 首次設定

1. 開啟擴充功能設定頁。
2. 檢查受控網站清單。預設包含 `zhihu.com`。
3. 依需要新增或移除網域。
4. 設定單次瀏覽限制、單一網站每日限制、所有受控網站每日總限制。
5. 選擇模式。
6. 啟用計時和攔截。
7. 儲存設定。

## 模式說明

- 溫和：單次瀏覽達到限制後關閉目前受控分頁。
- 攔截：達到限制後把受控網站跳轉到擴充功能內建攔截頁。
- 嚴格：執行攔截行為，並延遲放寬限制的設定變更。

## 工具列彈窗

點擊擴充功能圖示可以查看即時用量：

- 單次瀏覽：目前連續瀏覽時長。
- 目前網站今日：目前受控網站今天累計時長。
- 所有受控網站今日：所有受控網站今天累計時長。

## 通知提醒

擴充功能會在接近限制時發送 Chrome 通知，包括 30、20、10 秒和最後幾秒的倒數提醒。

## 安全說明

- 擴充功能預設關閉。
- 非受控網站不會被計時或攔截。
- `chrome://extensions` 等瀏覽器內部頁面不會被攔截。
- 使用資料只保存在本機瀏覽器儲存空間中。
