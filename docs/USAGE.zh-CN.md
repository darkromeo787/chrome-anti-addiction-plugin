# 使用帮助

## 安装未打包扩展

1. 打开 `chrome://extensions`。
2. 开启开发者模式。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目文件夹。
5. 如果需要快捷查看状态，可以把扩展固定到工具栏。

## 首次设置

1. 打开扩展设置页。
2. 检查受控网站列表。默认包含 `zhihu.com`。
3. 按需添加或移除域名。
4. 设置单次浏览限制、单个网站每日限制、所有受控网站每日总限制。
5. 选择模式。
6. 启用计时和拦截。
7. 保存设置。

## 模式说明

- 温和：单次浏览达到限制后关闭当前受控标签页。
- 拦截：达到限制后把受控网站跳转到扩展内置拦截页。
- 严格：执行拦截行为，并延迟放宽限制的设置变更。

## 工具栏弹窗

点击扩展图标可以查看实时用量：

- 单次浏览：当前连续浏览时长。
- 当前网站今日：当前受控网站今天累计时长。
- 所有受控网站今日：所有受控网站今天累计时长。

## 通知提醒

扩展会在接近限制时发送 Chrome 通知，包括 30、20、10 秒和最后几秒的倒计时提醒。

## 安全说明

- 扩展默认关闭。
- 非受控网站不会被计时或拦截。
- `chrome://extensions` 等浏览器内部页面不会被拦截。
- 使用数据只保存在本地浏览器存储中。
